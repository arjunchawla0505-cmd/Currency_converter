# Simple Currency Converter 💱

A lightweight, real-time currency conversion web application built with Python and Streamlit. The application fetches live, accurate exchange rates using the free ExchangeRate-API.

## 🚀 Live Demo

You can try out the deployed application here: 
👉 **[Live Streamlit App](https://arjunchawla0505-cmd-currency-converter-app-kfywvg.streamlit.app/)** *(Replace this link with your actual Streamlit deployment URL)*

---

## ✨ Features

- **Live Data:** Fetches real-time exchange rates directly from an API fallback.
- **Dynamic Conversion:** Instantly calculates conversions between major global currencies (USD, EUR, GBP, JPY, CAD, INR).
- **Clean UI:** A minimal, user-friendly interface that processes conversions at the click of a button.

---

## 🛠️ Tech Stack

- **Frontend/Framework:** Streamlit
- **Language:** Python 3
- **API Integration:** Requests library via [ExchangeRate-API](https://www.exchangerate-api.com/)
